Introduction

C'est quoi la différence entre le web et l'internet ?

Internet est un support de communication général, tandis que le Web est une application d'Internet inventée plusieurs années après.

Le principe du World Wide Web est de:

permettre l'accès à des documents de différentes natures situés sur des machines accessibles par Internet et reliés entre eux par un mécanisme de lien.

Une URL :

Une URL est le nom unique de la page, qui décrit l'emplacement d'une ressource et permet d'y accéder à partir d'un lien hypertexte.

Différence entre une page web et un site web

Une page web :

Une page web est un élément multimédia qui décrit ses éléments en utilisant un langage de description stocké dans un fichier HTML. On peut y accéder en utilisant un navigateur et elle est localisée sur Internet à l'aide d'une adresse (URL).

Un site web :

Un site web est un regroupement de pages web autour d'une thématique commune, accessibles à partir d'une page d'accueil. Ces pages sont stockées sous forme d'arborescence de dossiers et de fichiers.

Aspects techniques du Web

Les 3 mécanismes de base du Web sont :

  • URL : Permet l'accès à des ressources avec localisation via le protocole DNS
  • HTTP : Protocole client-serveur sans état
  • HTML : Langage de description de pages Web avec texte, image, liens et programmation déclarative

Création d'une page web HTML

HTML : Langage qui décrit des pages web.

Une page HTML est :

  • Un fichier texte structuré avec des balises
  • Interprété dans un navigateur web
  • Créé dans un éditeur de pages web ou de texte

Les principes de langage HTML :

Les balises:

Un fichier HTML doit avoir au minimum ces 4 balises :

  • <html> ... </html> : le début et la fin
  • <head> ... </head> : entête
  • <title> ... </title> : titre
  • <body> ... </body

    Les titres et paragraphes:

  • <hn> ... <hn> : Titre de niveau n :1 à 6.
  • <p> ... <p > : paragraphe.
